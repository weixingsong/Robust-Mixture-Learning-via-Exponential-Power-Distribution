# Robust-Mixture-Learning-via-Exponential-Power-Distribution

This repository contains all the R programs for the simulation studies in the paper "Adaptive Robust Mixture Learning via Exponential Power Errors".
There are three folders:

1. The folder "Simulation 1" has two files:

    (1.1). label_switching.R: align the parameter estimates with their true values
   
    (1.2). mixregEP_Simulation1: reproduce Figures 2,4 and Tables 1-2 in Simulation 1 using mixregEP procedure
   
    (1.3). mixregT_Simulation1: reproduce Figures 3,5 and Tables 1-2 in Simulation 1 using mixregT procedure
   
3. The folder "Simulation 2" has four files:

    (2.1). label_switching.R: aligning the parameter estimates with their true values
   
    (2.2). leverage_detection.R: detecting x-outliers
   
    (2.3). mixregEP_Simulation2: reproduce Tables 3,4,5,6 in Simulation 2 using mixregEP procedure
   
    (2.4). mixregT_Simulation1: reproduce Tables 3,4,5,6 in Simulation 2 using mixregT procedure

5. The folder "Simulation 3" has two files:

    (3.1). tonedata.txt: the tone dataset
   
    (3.2). tonedata.R: reproduce Figures 6 and 7.
